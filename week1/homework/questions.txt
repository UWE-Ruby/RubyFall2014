1. What is an object?

A: Objects are encapsulations of data (attributes etc..) and behavior (methods) that serve to represent objects or things just as they would in the real world. An example the book uses is an object representing a song. 

2. What is a variable?

A: Variables are names/references to objects. Variables can be thought as labels for the objects they contain. 

3. What is the difference between an object and a class?

A: Classes are templates that objects can created from. Any object can be said to be instance of class that was used to instantiate it.
